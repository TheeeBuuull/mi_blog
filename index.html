Práctica: Despliegue de un sitio estático

Inicio Práctica

Bienvenido

Esta práctica te guiará en la creación y publicación de una pequeña web estática usando GitHub Pages.

Objetivo de la práctica

El propósito es que experimentes con control de versiones, personalización de un sitio web y publicación automática. También verás la importancia de la documentación y del uso de herramientas como Git, que facilitan la colaboración y el mantenimiento del proyecto.
